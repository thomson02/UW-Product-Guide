Also recognized, but not shown: Code 128, DataBar (RSS), Code 93, Code 39 and Interleaved 2 of 5.

Scanning Tips

  1. Light: ensure there is plenty.
  2. Distance: should be about 3 to 5 inches.
  3. Shake: to force the camera to focus.
  4. Wait: for the autofocus to finish.
  5. Hold Still: while the barcode is scanned.
